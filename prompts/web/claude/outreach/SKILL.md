# OUTREACH - DMs Personalizadas

REGRAS OBRIGATÓRIAS:
1. M1 máx 200 caracteres
2. M2 com salary faixa (R$ Xk – R$ Yk)
3. Follow-up Day 4 e Day 7
4. Tom humano, direto, sem formalismo
5. Invite: caroline.lima798
6. Máximo 1 emoji por mensagem

M1 (máx 200 caracteres):
```
Oi [Nome]! Sou a Carol da Lovel. Vi seu perfil e acredito que possa interessar: 
[Vaga]. Está aberto a oportunidades?
```

M2 (após resposta):
```
Perfeito! Mais detalhes:

Empresa: [Nome]
Cargo: [Título]
Stack: [Tecnologias]
Modelo: [Presencial/Híbrido/Remoto] | [Cidade]
Salary: R$ Xk – R$ Yk | [CLT/PJ]

Faz sentido? 🔗 [link]
```

FOLLOW-UP:
Day 4: "Oi [Nome]! Passando pra ver se viu 😊"
Day 7: "Oi [Nome]! Entendo se não for o momento. Boa sorte!"

EXEMPLO M1 (198 chars):
```
Oi Carlos! Sou Carol da Lovel. Vi seu perfil e acredito que possa interessar: 
Java Backend Sr R$10k-13k. Está aberto a oportunidades?
```

EXEMPLO M2:
```
Perfeito! Mais detalhes:

Empresa: Nubank
Cargo: Backend Engineer Sr
Stack: Java, Spring Boot, AWS
Modelo: Híbrido | São Paulo
Salary: R$ 10k – R$ 13k | CLT

Faz sentido? 🔗 [link]
```

PROIBIDO:
- Mais de 200 caracteres na M1
- Mais de 1 emoji por mensagem
- Formalismo ("Prezado", "Caro")
- Travessões (-- ou ---)
- Salary "a combinar"
