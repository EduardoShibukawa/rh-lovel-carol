# OUTREACH - DMs Personalizadas & 3x3 Rule

## Overview

Skill para criar mensagens de outreach personalizadas seguindo a 3x3 Rule de Stacy Zapar (máx 200 caracteres na M1) e estrutura de P-C-T-F de Allie Miller.

## Autoridades Aplicadas

- **Stacy Zapar**: 3x3 Rule (máx 200 chars, 3 pontos personalização, follow-up 3 dias)
- **Allie Miller**: P-C-T-F (Persona, Context, Task, Format)
- **Aaron Ross**: Closing CTA ("Faz sentido?")
- **Lou Adler**: Impacto de 90 dias na descrição
- **Lovel**: Tom humano, transparência salarial

## Workflow

### Passo 1: Verificar Dados do Candidato
Se não houver dados do perfil:
- Usar placeholders como `[DETALHE REAL]`
- Exemplos do que buscar:
  - Última empresa/função
  - Projeto ou resultado específico
  - Stack ou área de atuação

### Passo 2: Gerar M1 (Máx 200 caracteres)
Estrutura:
```
Oi [Nome]! Sou [Carol], parceira da Lovel. Vi seu perfil e acredito que possa lhe interessar 
uma oportunidade em [Empresa Setor] focada em [Impacto/Tecnologia]. 
Está avaliando novas oportunidades?
```

**Contagem: [X]/200 caracteres**

### Passo 3: Gerar M2 (Após resposta)
Estrutura:
```
Perfeito! Vou te compartilhar mais detalhes 😊

Empresa: [Nome]
Cargo: [Título + Senioridade]
Atuação/Stack: [Atividades core ou stack]
Modelo: [Presencial/Híbrido/Remoto] | [Cidade/UF]
Remuneração: [R$ Xk – R$ Yk] | [CLT ou PJ]
Diferencial: [1 frase de impacto]

Faz sentido?

🔗 [Link Lovel com invite=caroline.lima798]
```

### Passo 4: Follow-up (3x3 Rule)
- **Day 4**: "Oi [Nome]! Passando pra ver se conseguiu dar uma olhada 😊 Fico à disposição!"
- **Day 7**: "Oi [Nome]! Entendo que o momento pode não ser ideal agora. Fico por aqui se surgir algo no futuro 😊 Boa sorte!"

## Limitations

- **PROIBIDO**: Exceder 200 caracteres na M1
- **PROIBIDO**: Usar travessões (—) ou separadores (---)
- **PROIBIDO**: Usar mais de 1 emoji por mensagem
- **PROIBIDO**: Frases genéricas ("Hope this finds you well", "boa tarde")
- **SEMPRE**: Incluir faixa salarial explícita na M2
- **SEMPRE**: Incluir link com invite=caroline.lima798

## Validation

- [ ] M1 com máximo 200 caracteres?

- [ ] Follow-ups Day 4 e Day 7?
- [ ] Tom humano, sem formalismo?
- [ ] Invite parameter presente?
