# OODA Orchestrator Skill

This skill is responsible for managing the Observe-Orient-Decide-Act (OODA) loop, coordinating research, strategy formulation, and execution phases for various tasks. It delegates work to specialized sub-agents and skills, ensuring adherence to project mandates and efficiency.
